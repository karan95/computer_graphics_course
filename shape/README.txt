Computer Graphics I - Programming Assignment 1: 2D Drawing

Implemented using JavaScript, Html5, Css3

Objective: Implement 2D primitive rendering algorithms to draw shape like line, circle, and ellipse.

Project provides the interactive UI to draw shapes. User must provide input like (x, y) point, radius, width and height
to draw particular shape.

This algorithms can be used to draw complex design like graphs or data driven document patterns.
